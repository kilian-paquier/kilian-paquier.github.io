Pointeuse

Lors de ma première année à Polytech Tours, afin de valider mes compétences en développement orienté objet avec Java, j'ai développé une pointeuse qui permet aux employés d'une entreprise d'enregistrer leurs heures de pointage. Le logiciel communique via TCP/IP entre les pointeuses et l'application centrale, celle-ci permettant de gérer les employés (ajout, modification, suppression), les départements et les managers. Enfin, si les pointeuses ne peuvent communiquer avec l'application centrale, elles enregistrent les pointages dans un fichier JSON qui sera par la suite transmis lorsque la communication sera de nouveau fonctionnelle.
